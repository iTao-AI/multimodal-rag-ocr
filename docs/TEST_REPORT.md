## ❌ 测试失败项

### 失败 1: 大文件上传 (> 50MB)

**测试用例**: PDF 上传 - 大文件  
**测试文件**: large_manual.pdf (65MB)  
**预期结果**: 上传成功，处理时间 < 5 分钟  
**实际结果**: ❌ 超时 (5 分钟后连接断开)  
**错误日志**:
```
[ERROR] Upload timeout after 300s
File size: 65MB
```

**根本原因**: 
- uvicorn 默认超时时间 300 秒
- 大文件处理时间过长

**修复方案**:
```python
# 修改 uvicorn 配置
uvicorn.run(
    "main:app",
    host="0.0.0.0",
    port=8000,
    timeout_keep_alive=600,  # 延长到 10 分钟
)
```

**优先级**: P1  
**预计修复时间**: 1 小时

---

### 失败 2: 并发对话压力测试

**测试用例**: 对话问答 - 并发测试  
**测试场景**: 50 个并发对话请求  
**预期结果**: 所有请求成功，平均响应时间 < 2 秒  
**实际结果**: ❌ 部分失败 (10 个请求返回 503)  
**错误日志**:
```
[ERROR] Connection pool exhausted
Active connections: 100/100
```

**根本原因**:
- 数据库连接池大小限制 (100)
- 并发请求超出连接池容量

**修复方案**:
```python
# 增加连接池大小
engine = create_engine(
    DATABASE_URL,
    pool_size=200,      # 增加到 200
    max_overflow=50,    # 允许溢出 50
    pool_pre_ping=True, # 连接前检查
)
```

**优先级**: P1  
**预计修复时间**: 2 小时
