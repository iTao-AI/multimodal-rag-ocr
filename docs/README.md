## 📝 文档规范

### 命名规范

- ✅ 使用大写：`API.md`, `README.md`
- ✅ 使用下划线：`USER_GUIDE.md`
- ❌ 避免空格：`user guide.md`
- ❌ 避免中文：`用户指南.md`

### 格式规范

- 使用 Markdown 语法
- 包含目录导航
- 添加版本和日期
- 使用表格展示对比
- 代码块标注语言

### 更新规范

- 重大变更更新版本号
- 修改后更新日期
- 在 CHANGELOG.md 记录变更
- 保持链接有效性
