### 数据模型

#### 文档模型

```python
class Document:
    id: str              # 文档 ID
    filename: str        # 文件名
    file_path: str       # 文件路径
    file_size: int       # 文件大小 (bytes)
    page_count: int      # 页数
    upload_time: datetime # 上传时间
    status: str          # 状态：pending/processing/completed/failed
    metadata: dict       # 元数据
```

#### 向量模型

```python
class Vector:
    id: str              # 向量 ID
    collection: str      # 集合名称
    vector: List[float]  # 向量数据 (1024 维)
    chunk_text: str      # 文本内容
    chunk_id: str        # 块 ID
    filename: str        # 来源文件
    page: int            # 页码
    metadata: dict       # 元数据
```

#### 对话模型

```python
class Conversation:
    id: str              # 会话 ID
    user_id: str         # 用户 ID
    collection: str      # 知识库
    messages: List[Message] # 消息列表
    created_at: datetime # 创建时间
    updated_at: datetime # 更新时间
```
